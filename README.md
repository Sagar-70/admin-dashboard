# 🚀 Admin Dashboard (Flask Project)

A simple and responsive **Admin Dashboard Web Application** built using **Flask (Python)** with a clean UI and secure login system.

---

## 📌 Features

* 🔐 Admin Login System
* 📊 Interactive Dashboard
* 🎨 Responsive UI (HTML, CSS, JavaScript)
* 🧭 Sidebar Navigation
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** Not used (Static authentication)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Sagar-70/admin-dashboard.git
```

2. Go to project folder:

```bash
cd admin-dashboard
```

3. Create virtual environment:

```bash
python -m venv venv
```

4. Activate environment:

```bash
venv\Scripts\activate
```

5. Install dependencies:

```bash
pip install -r requirements.txt
```

6. Run the app:

```bash
python app.py
```

7. Open in browser:

```bash
http://127.0.0.1:5000/
```
